# High-Engagement, Low-Discovery User Research Report
## Introduction
This report synthesizes the results of five user interviews with high-engagement, low-discovery Spotify users. The goal of this research is to identify key themes and frustrations, user workarounds, root cause hypotheses, and opportunities for an AI solution to improve the music discovery experience.

## Key Themes & Frustrations
The following themes and frustrations emerged from the user interviews:

* **Lack of control over music recommendations**: Users feel that the current algorithm is too safe and doesn't allow them to specify their mood or genre preferences.
* **Insufficient discovery of new music**: Users are not discovering new artists or songs through the current discovery features, such as Discover Weekly and Radio.
* **Over-reliance on familiar music**: The algorithm prioritizes familiar music, leading to a lack of diversity in recommendations.
* **Limited context for recommendations**: Users want to be able to provide more context about their mood, activity, or preferences to receive more relevant recommendations.

## User Workarounds
To address the limitations of the current algorithm, users have developed the following workarounds:

* **Searching for playlists on Reddit**: Users search for playlists on Reddit to discover new music and artists.
* **Going incognito on Spotify**: Users go incognito to receive recommendations that are not based on their listening history.
* **Using other music streaming services**: Some users have tried other music streaming services in search of better discovery features.

## Root Cause Hypothesis
Based on the user interviews, the following root cause hypotheses can be formulated:

* **Over-reliance on collaborative filtering**: The current algorithm may be too reliant on collaborative filtering, which prioritizes familiar music and doesn't allow for sufficient discovery of new artists and songs.
* **Lack of contextual understanding**: The algorithm may not have a deep understanding of the user's context, such as their mood, activity, or preferences, leading to irrelevant recommendations.
* **Insufficient exploration-exploitation trade-off**: The algorithm may be too focused on exploitation (playing familiar music) and not enough on exploration (discovering new music), leading to a lack of diversity in recommendations.

## Opportunity Area for an AI Solution
An AI-powered solution can address the limitations of the current algorithm by:

* **Incorporating natural language processing (NLP) to understand user context**: An AI-powered solution can use NLP to understand the user's mood, activity, or preferences and provide more relevant recommendations.
* **Using reinforcement learning to balance exploration and exploitation**: An AI-powered solution can use reinforcement learning to balance the trade-off between exploration and exploitation, leading to a more diverse and relevant set of recommendations.
* **Integrating knowledge graph-based recommendations**: An AI-powered solution can use knowledge graph-based recommendations to provide more accurate and diverse recommendations based on the user's listening history and preferences.

By addressing the key themes and frustrations, user workarounds, and root cause hypotheses, an AI-powered solution can improve the music discovery experience for high-engagement, low-discovery users and provide a more personalized and relevant set of recommendations.
